# YT2MPEG
The simplest app to download MP3 or MP4 files.

# Running
Just run the "Run.bat" file and it automatically installs all other requirements OTHER THAN FFMPEG

# Requirements
 * FFMPEG
 * Python 3.10+ (preferrably 3.14.x)

# Installation
 Just install the requirements.txt and make sure ffmpeg is installed and added to path.
